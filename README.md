# SuperStore

## Description

A webshop frontend written in React, HTML and CSS. The webshop use a REST API to fetch products and images.

## Getting Started

### Installing

1. Clone the repo:

```bash
git clone git@github.com:toreandre/superStore.git
```

2. Install the node modules

```bash
npm install
```

### Running

```bash
npm run start
```
